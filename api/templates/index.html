Preview Generator

The following endpoints are available:

  • POST to /preview?filename=myfile.pdf&width=150&height=200&need_crop=0&extension=jpeg. The body should contain data

      Query params:
    • width (int) - необходимая ширина
      По умолчанию: 256

    • height (int) - необходимая высота
      По умолчанию: 256

    • need_crop (int[0-1]) - необходимо подрезать изображение до указанных размеров
      По умолчанию: 1

    • extension (str['jpeg', 'pdf', 'html', 'text']) - возвращаемый файл.
      По умолчанию: 'jpeg'
